# Proyecto: estructurasRepeticionJava
_Ejercicios prácticos de estructuras de repetición en Java_

## Descripción
Este proyecto contiene una serie de ejercicios en **Java** que aplican las **estructuras de repetición** (`for`, `while`, `do-while`) para resolver problemas básicos de lógica y programación.  
Forma parte de un portafolio académico y profesional orientado a la práctica de **estructuras de control de flujo**.

---

## Objetivos
- Comprender y aplicar bucles simples y anidados.
- Utilizar correctamente las estructuras `for`, `while` y `do-while`.
- Desarrollar programas modulares y bien documentados con **código limpio**.
- Practicar entrada y salida de datos con `Scanner` y `System.out`.

---

## Contenido del proyecto

| Ejercicio        | Clase                  | Descripción |
|------------------|------------------------|-------------|
| TablaMultiplicar | `tablaMultiplicar.java` | Genera la tabla de multiplicar de un número ingresado. |
| Arreglo          | `arreglo.java`          | Maneja operaciones básicas con arreglos. |
| Carrito          | `carrito.java`          | Simula un carrito de compras con repetición. |
| PromedioNotas    | `promedioNotas.java`    | Calcula el promedio de notas ingresadas por el usuario. |
| RegistrosVentas  | `registrosVentas.java`  | Gestiona registros de ventas usando estructuras repetitivas. |

---

## Contenido temático
- **Repetición controlada** → `for`
- **Repetición condicional** → `while`
- **Repetición post-condicional** → `do-while`
- **Bucles anidados**
- **Acumuladores y contadores**
- **Procesamiento de arreglos y colecciones**
- **Simulación de procesos repetitivos**
